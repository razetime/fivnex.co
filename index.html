Our projects

The projects that make Fivnex a thing.

MEML

MEML is an open-source, Lisp-like, web-development language that transpiles into HTML, CSS, and even JavaScript.

Keter

A general purpose programming language built for speed, readability, and user-friendliness. Inspired by V/Go/Rust.

Potabi

A FreeBSD-based operating system built for ease of use in development, office, and home workspaces, for people to enjoy.

Cherry

A GNOME3 desktop environment fork to build an easy-to-use and user-friendly desktop environment for all *nix systems.

Fivnex Computers

Desktop computers running open-source operating systems. Built for people who want power + ease of use in their desktop.

Wedeke

A simple code editor for MEML and Keter. Designed to work fast, show immediate results, and easy for testing things.
